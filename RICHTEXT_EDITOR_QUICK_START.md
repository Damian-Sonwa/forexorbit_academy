# Quick Start: Using the Lesson Content Editor

## For Instructors

### What You Can Do

The new Lesson Content Editor allows you to format lesson content with:
- **Text Formatting:** Bold, italic, underline, strikethrough
- **Headings:** Use H1, H2, H3, H4 for section titles
- **Lists:** Bullet points and numbered lists
- **Advanced:** Links, images, tables, and code blocks
- **Alignment:** Center, right, or justify text
- **Fonts:** Choose font family and size

### How to Use (Step-by-Step)

#### Step 1: Navigate to a Lesson
1. Go to your lesson page
2. You'll see the video player at the top

#### Step 2: Open the Editor
1. Look below the video player
2. You'll see a section titled "Lesson Content Editor"
3. Click the **"Edit Content"** button (amber/orange button)
4. The rich text editor will expand

#### Step 3: Write Content
1. Click in the editor area
2. Type your content
3. Use the toolbar buttons to format text:
   - **B** = Bold
   - **I** = Italic
   - **U** = Underline
   - **S** = Strikethrough

#### Step 4: Use Advanced Features
- **Headings:** Select text → Click "Paragraph" dropdown → Choose H1, H2, H3, or H4
- **Lists:** Select text → Click bullet or number icon
- **Links:** Click link icon → Enter URL
- **Images:** Click image icon → Enter image URL
- **Tables:** Click table icon → Create table
- **Code:** Select code → Click code block icon
- **Alignment:** Click alignment buttons (left, center, right, justify)

#### Step 5: Save Content
- **Automatic:** Content auto-saves every 30 seconds (you'll see "Saving..." then "Saved")
- **Manual:** Click **"Save Lesson Content"** button for immediate save

#### Step 6: Preview
1. Click **"Hide Editor"** to close
2. Scroll down to see how the content looks
3. The content will be formatted exactly as you see it in the editor

### Important Notes

✅ **Your content auto-saves every 30 seconds** - You don't have to manually save
✅ **You can edit anytime** - Just click "Edit Content" again
✅ **Students see a clean version** - No editing tools visible to them
✅ **Formatting is preserved** - All bold, headings, etc. display correctly to students

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+B or Cmd+B | Bold |
| Ctrl+I or Cmd+I | Italic |
| Ctrl+U or Cmd+U | Underline |
| Ctrl+Z or Cmd+Z | Undo |
| Ctrl+Y or Cmd+Y | Redo |

### Toolbar Buttons (Left to Right)

1. **Undo/Redo** - Undo or redo changes
2. **Format** - Change text format (paragraph, headings, etc.)
3. **Bold/Italic/Underline/Strikethrough** - Text styling
4. **Text Color** - Change text color
5. **Alignment** - Align text left, center, right, or justify
6. **Lists** - Create bullet or numbered lists
7. **Remove Formatting** - Remove all formatting from selected text
8. **Link** - Insert or edit a link
9. **Image** - Insert an image (provide URL)
10. **Media** - Insert video or media
11. **Table** - Create a table
12. **Code Block** - Insert formatted code
13. **Fullscreen** - Edit in fullscreen mode
14. **Preview** - Preview how content will look
15. **Code View** - View raw HTML code
16. **Help** - Get help

### Tips & Best Practices

#### ✅ DO's
- ✅ Use headings to organize your content
- ✅ Use lists for step-by-step instructions
- ✅ Add images to illustrate concepts
- ✅ Keep text concise and scannable
- ✅ Use bold for important terms
- ✅ Test your content by hiding the editor and scrolling

#### ❌ DON'Ts
- ❌ Don't copy formatting from Word (may include hidden styles)
- ❌ Don't use too many different fonts
- ❌ Don't use too many colors
- ❌ Don't forget to save important changes
- ❌ Don't use huge font sizes for body text

### Troubleshooting

#### "Editor doesn't appear"
- Try refreshing the page
- Make sure you're logged in as an instructor
- Check browser console for errors (F12)

#### "Content isn't saving"
- Check your internet connection
- Look for error message in red
- Try clicking "Save Lesson Content" manually
- If it still fails, refresh and try again

#### "Formatting looks wrong"
- Make sure you're using the format dropdown (not just bold)
- Try removing formatting and re-applying
- Check the preview mode
- Refresh the page

#### "Image won't display"
- Make sure the image URL is correct
- Use HTTPS URLs (not HTTP)
- Check that the image exists at that URL
- Try a different image URL

### FAQs

**Q: Can I undo changes?**
A: Yes! Use Ctrl+Z (or Cmd+Z on Mac) or click the Undo button

**Q: Will students see the editing toolbar?**
A: No! Students only see the formatted content. No editor is visible to them.

**Q: How often is my content saved?**
A: Automatically every 30 seconds, or click "Save Lesson Content" for immediate save

**Q: Can I copy content from Word?**
A: Yes, but you may need to clean up formatting. Use "Remove Formatting" if needed.

**Q: Can I embed videos?**
A: Yes, click the "Media" button and paste the video URL (YouTube, Vimeo, etc.)

**Q: What file types can I insert?**
A: You can link to PDFs, images (PNG/JPG/GIF), and videos

**Q: How much content can I add?**
A: Theoretically unlimited, but keep it reasonable (under 100,000 characters recommended)

**Q: Is my content backed up?**
A: Yes! It's automatically saved to the database and can be accessed anytime
